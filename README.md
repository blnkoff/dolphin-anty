# Installing dolphin-anty
To install `dolphin-anty` from PyPi, you can use that:

```shell
pip install dolphin-anty
```

To install `dolphin-anty` from GitHub, use that:

```shell
pip install git+https://github.com/blnkoff/dolphin-anty.git
```
